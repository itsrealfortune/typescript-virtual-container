    Interface PackageDefinition

    Metadata and behaviour definition for a single package in the registry.

    Used both for the built-in registry entries and for consumer-supplied custom packages. files are written to the VFS on install(), and onInstall / onRemove hooks allow arbitrary VFS mutations.
